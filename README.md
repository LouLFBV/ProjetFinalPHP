# ProjetFinalPHP

Pour la database du projet, il faudra installer XAMPP, sur le control panel il faut faire start pour apache et mysql. Ensuite une fois dans VSCODE, installer SQLTools, add new connection. Driver Mysql/MariaDb si vous l'avez pas, installez l'extension puis choissisez mysql. Pour les datas :
database name : XAMPP_Local
DataBase : mysql
username : root
password mode : empty password
port : 3306
server : localhost
